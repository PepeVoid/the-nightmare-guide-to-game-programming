# The Nightmare Guide to Game Programming
The guide you will hate but you will need to learn game programming.

## Requirements

- [Rust](https://www.rust-lang.org/es)

- [mdBook](https://rust-lang.github.io/mdBook/index.html)

## Installation

Clone the repository:

```console
git clone https://github.com/aburronDev/the-nightmare-guide-to-game-programming.git
```

## Usage

1. With [Rust](https://www.rust-lang.org/es) and [mdBook](https://rust-lang.github.io/mdBook/index.html) installed:

```console
mdbook serve
```

2. Open a new browser window and search for `http://localhost:3000`
